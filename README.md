# Monk's Theater of the Mind
Add-On Module for Foundry VTT
For those times when you don't need to move tokens around the canvas, but still want combat to be entertaining,

## Installation
Simply use the install module screen within the FoundryVTT setup

## Usage & Current Features
Use the theater of the mind button to open up the window.
You can use the controls on the page to adjust combat.
Use the setting icon to change the background
